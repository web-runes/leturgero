This guide gets you started with the Leturgerð CLI.

## Using the CLI

In your terminal, execute the CLI using your preferred package manager in your project directory:

<PackageManagerTabs>
  <Fragment slot="npm">
    ```sh
    npx @leturgero/cli@latest
    ```
  </Fragment>
  <Fragment slot="pnpm">
    ```sh
    pnpx @leturgero/cli@latest
    ```
  </Fragment>
  <Fragment slot="yarn">
    ```sh
    yarn @leturgero/cli@latest
    ```
  </Fragment>
</PackageManagerTabs>

That's it! The CLI will guide you (or your agent) through.

## Next steps

Once you're done with the CLI, follow our guides on how to hook fonts in your project.
